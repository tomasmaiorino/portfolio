Sumary

Responsible for developing new features and maintenance for a brasilian ecommerce using Oracle ATG Commerce 10 also responsible for developing and maintenace microservices (Spring boot and rest services) that are consumed by the store. Analyzing and fixing problems in production environment. Managed Oracle Databases. Worked on the planning and implementation of approvals, holding meetings with users to define and gather system requirements.

Main Projects

  • Change the default ATG's recover password workflow to return a link to user change his password instead automaticly change the password.

  • Send some order information to an external service during payment pipeline through JMS (JRabbit) integration.

  • Update user's contact information in the confirmation page in order to help to contact the user if some problem happens with his order.

  • Use BCC's Content Group to choose which product data will be send to the external service.
